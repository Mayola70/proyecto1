Los Puntos a Dos agujas...

Para empezar a tejer siempre decimos que se necesita tan solo agujas y un ovillo. Y es cierto, es tan simple como eso, porque lo primero que vamos a aprender es a montar puntos, tejer un punto fácil como el santa clara o punto bobo y tejer todo recto para luego cerrar. Pero en la medida en que nos enganchemos con el tejido, y comencemos a tejer diferentes tipos de prendas de mayor complejidad, iremos necesitando añadir más elementos a nuestra lista de material para tejer, y sobre eso les quiero hablar en este post.

¿Qué se necesita para comenzar a tejer?

Si quieres aprender directamente a tejer a dos agujas, ¡también es perfectamente posible! Para aprender a tejer punto solo necesitaras dos agujas, un ovillo, y muchas ganas. Algunos consejos para comenzar a tejer es utilizar una lana gruesa, con un punto sencillo para que en poco tiempo puedas tener tu primera prenda lista, ¡te dará motivación para seguir aprendiendo!
